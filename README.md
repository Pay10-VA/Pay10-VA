# Recent Personal Projects

### **CoDoc — Real-Time Collaborative Document Editor**

_October – November 2025 · TypeScript · Node.js · Express · Next.js · TailWind CSS · PostgreSQL_

Full-stack collaborative document editor built with Next.js, Express, and PostgreSQL. Implements CRDT-based synchronization via Y.js and TipTap for conflict-free simultaneous editing, with Socket.IO powering live comments, file management, and real-time collaborator presence.

### **EventStream — Real-Time Event Analytics Platform**

_March – April 2026 · Java · Spring Boot · Kafka · Redis · MongoDB · Docker_

Real-time event ingestion and analytics platform that captures user behavior events, streams them through Kafka with retry and DLQ support, persists them to MongoDB, and tracks live analytics using a Redis sliding window algorithm.

### **EventStream MCP Server**

_April 2026 · Python · Fast MCP_

MCP server built with FastMCP that exposes EventStream's analytics API as Claude-callable tools. Enables natural language querying of funnel drop-off rates, top products, per-user event counts, and real-time activity.
